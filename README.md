# Seaside Restaurant & Bar - Website Clone

A clone of the [Seaside Restaurant & Bar](https://www.seasiderestaurantbar.com/) website.

## Structure

- **index.html** - Main HTML structure with all sections
- **styles.css** - Complete styling including responsive design
- **script.js** - Smooth scrolling and header scroll effects

## Sections

1. **Header** - Logo, social links (Instagram, DoorDash), Online Order button
2. **Hero** - Full-screen restaurant interior with "The Best Clam Chowder" headline
3. **About Us / Our Story** - Two-column layout with text and food image, wavy dividers
4. **Full Bar** - Parallax-style section with bar background
5. **Our Work** - Team and experience description
6. **Menu** - "Glimpse of Our Menu" with food image grid
7. **Footer** - Copyright and review links
